# Y - The (very lazy) shortcuts
If you don't like to write a lot of code and want to make your code hard to read for others,
then Dream-Yaml shortcuts should be perfect for you.

| Shortcut | Equivalent |
| :-----: | :-----: |
|`new Y();`|`new DreamYaml();`|
|`k();`|`put()`|
|`dv();`|`setDefaultValues()`|
|`v();`|`setValues()`|
